template<DebugLevels DebugLevel_, OutInterfaces OutInterface_>
class rgbd::LoggableInterface< DebugLevel_, OutInterface_ >

Interface class to add log capabilities to class.

Parameters
DebugLevel_Define displayed messages. Debug display all the messages; Warning warnings and errors; Error just the errors ; and Null nothing.
OutInterfacesDefine how messages are managed. Cout mode uses standart std::cout and LogManager uses rgbd_tools LogManager util which writes in the log file too.
