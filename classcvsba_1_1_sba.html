Run sba algorithm. We are using the same interface defined in cv::LevMarqSparse.

Let N and M denote the number of object points and the number of cameras.

Parameters
pointsN x 3 object points
imagePointsM x N x 2 image points for each camera and each points. The outer vector has M elements and each element has N elements of Point2d .
visibilityM x N x 1 visibility matrix, the element [i][j] = 1 when object point i is visible from camera j and 0 if not.
cameraMatrixM x 3 x 3 camera matrix (intrinsic parameters) 3 x 3 camera matrix for each image
distCoeffsM x 5 x1 distortion coefficient for each image
RM x 3 x 3 rotation matrix for each image
TM x 3 x 1 translation matrix for each image
Returns
average reprojection error of the optimized solution
Note
matrices can be either float or double, they are internally changed.
